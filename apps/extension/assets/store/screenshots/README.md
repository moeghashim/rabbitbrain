# Screenshot Checklist

Capture these from the built extension on a desktop Chrome profile after loading the unpacked extension:

1. Analyze button injected on a public post in the timeline
2. Inline analysis result with topic, summary, and concept chips
3. Tag editing and Save to Bookmarks state
4. Auth-required state after sign-in opens in a tab
5. Bookmark success state with the Bookmarks link visible

Target Chrome Web Store screenshot size: `1280x800`.
