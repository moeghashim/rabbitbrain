# Chrome Web Store Listing Draft

## Title
Rabbitbrain for X

## Category
Productivity

## Short Description
Analyze public X posts and save tagged insights to Rabbitbrain.

## Detailed Description
Rabbitbrain for X adds Rabbitbrain actions directly inside x.com. Analyze a public post inline, review the topic, summary, intent, and concept chips, then save tagged insights to your Rabbitbrain bookmarks without leaving the timeline.

Rabbitbrain for X requires a Rabbitbrain account connected with X. The extension is not affiliated with X Corp.

## Privacy Field Guidance
- Single purpose: Analyze public X posts and save tagged insights to Rabbitbrain.
- Website content handled: tweet URL, tweet text, and tweet metadata for the post you choose.
- User activity handled: bookmark tags and in-extension action state needed to resume sign-in.
- Personal identifiers handled: Rabbitbrain account identifiers needed to confirm sign-in.
- Remote code: No.
